# Cris Analytic

Sitio estatico minimo para publicar en GitHub Pages con el dominio `crisanalytic.com`.

## Archivos

- `index.html`: pagina principal
- `CNAME`: dominio personalizado para GitHub Pages
- `.nojekyll`: evita el procesamiento de Jekyll

## Publicacion

1. Crea un repositorio en GitHub y sube este contenido.
2. En GitHub, activa **Pages** desde la rama principal (`main`) y la carpeta raiz (`/`).
3. Configura el DNS de `crisanalytic.com` para apuntar a GitHub Pages.
